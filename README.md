# tracking-service


### 패키지 설치
``` 
$ npm install
```

### Config 설정
``` 
const config = {
  address: '',
  privateKey: '',
  httpEndpoint: 'http://octet-fullhistory-ropsten.hexlant.com:2020',
  accessKey: '',
  pickerServiceContractAddress: 'YOUR_CONTRACT_ADDRESS',
}
``` 

### 실행
``` 
$ npm start
```
